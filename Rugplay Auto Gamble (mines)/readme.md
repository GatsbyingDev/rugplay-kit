# Rugplay Auto Gamble (Tampermonkey Script)

**Rugplay Auto Gamble** is a Tampermonkey userscript that automates Rugplay’s mines feature. It automatically clicks tiles, starts games, cashes out, and can run multiple tiles at once. It also has a toggle interface with options like “25% wager” and multi-tile mode for faster gameplay.

> ⚠️ **Disclaimer:** Use responsibly. This script is for **educational and fun purposes**. Test on small accounts first. Do not share your account cookies or personal info.

---

## Features

* Auto-clicks tiles in Rugplay mines
* Auto-starts games and cashes out
* Options for 25% wager and multiple tiles at once
* Simple on-screen toggle interface
* Console logging for debugging and monitoring

---

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) for your browser.
2. Click **“Create a new script”** in Tampermonkey.
3. Copy the contents of `main.user.js` into the editor.
4. Save the script.
5. Visit [Rugplay.com](https://rugplay.com) and the script will inject controls at the bottom-right corner.

---

## Usage

1. Open Rugplay mines.
2. Toggle the **Auto Miner** button ON.
3. Check/uncheck options:

   * **25% wager** → Bet 25% automatically
   * **Multiple tiles** → Click 3 tiles per turn
4. The script will click tiles, start games, and cash out automatically.
5. Toggle OFF to stop the script.
